Introduction to Programing

HTML and the "Web"
Intro

The World Wide Web is made up of HTML or Hypertext Markup Language documents. Approximately 30 billion documents are "webbed" by hyperlinks. There are three major pieces that make up the web: a user with a computer and a browser that "reads" webpages,the internet (uses HTTP to make and send requests to servers), and servers, computers optimized to host the documents that make up the web.

HTML - Hypertext Markup Language

HTML is made up of four main components: text content (what you can see), markup (creates the "look" of the page), references to other documents (e.g. images and videos), and links to other pages.

Markup - Tags and Elements

Elements are made up of tags (both opening tags, and closing tags) with content between the tags. These tags instruct the computer to change the content between them in a particular way (e.g. making text bold or italics).

One type of tag, known as an attribute, can be used to create a link in the text (among other functions). For example: Practice HTML Here! Tags can also be used to insert images onto a page. Keep in mind that tags create elements that are either inline or block. Inline elements are on the line with text while block elements create an "invisible box" around the content.

Computers are Stupid

Computers read instructions literally. For example: if you use a bold opening tag and failed to use a closing tag, the computer would interpret that as you asking for everything after the opening tag to be in bold text.

Some General Concepts to Remember

1 - Webpages are text documents written in HTML and "read" by browsers.

2 - Coding is instructions written in computer language.

3 - Computers are stupid!

4 - Programmers can't remember everything. Don't be afraid to look things up!

5 - Basic HTML vocabulary: tag, element, attribute.

Creating Structure and Style
Elements within Elements and Boxes within Boxes

Whether we are talking about the HTML code itself or the boxes that make up the "structure" of a webpage, everything is layered.

Indenting: Waves and Trees

Looking at HTML code reveals a wave created by indenting the divisions (read: boxes). You can visually see how "deep" a box is by its level of indentation.

Note: It's a good idea to create this structure first (even on paper using the "box method") before muddying the water with content.
